## User Manual

To run the Data Labelling program, ensure you are have Java version 17 or higher installed on your machine and download the Jar package available ![here](https://github.com/S010MON/data-labelling/releases/download/v0.1/data-labelling-1.0.jar).

Open a terminal and navigate to the directory that contains the jar.  Execute the jar using the following command:

```
java -jar data-labelling-1.0.jar
```

You should be greeted by the welcome screen![Screenshot from 2022-03-24 19-40-50](https://user-images.githubusercontent.com/10490509/160142198-47175fdc-fa98-4264-8f03-bc1cf595f060.png)


To start running a test, click the green start button and use the mouse or trackpad to draw bounding boxes around objects within the image.
